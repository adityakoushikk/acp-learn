## Local development

1. **Start the Flask backend** (in one terminal):
   ```bash
   ./start-backend.sh
   ```
2. **Start the Next.js app** (in another terminal):
   ```bash
   pnpm dev
   ```
3. **Open in browser:** [http://localhost:3000](http://localhost:3000)
